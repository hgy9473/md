# DOS 命令使用笔记
---
## move

### 移动文件
```c
move tech/健壮代码.md coding-skill/健壮代码.md
```

### 移动文件夹
```c
move coding-skill tech
```

---
## mkdir

### 在指定文件夹新建文件夹
```c
mkdir tech\cesium
```

---
## rd

### 删除文件夹
```c
rd coding-skill
```

---
## 查看系统信息
```c
// 关于
winver
// 详细信息
msinfo32
```

## 查看文件夹结构
```c
tree
```

## 查看文件以及文件夹结构
```c
tree /f
```

##查看文件夹以及文件夹结构，并输出到文件
```c
tree /f >tree.txt
```
